# Challenge Final – Ecommerce WhatsApp API

API desarrollada con FastAPI que simula un vendedor de ecommerce
atendiendo consultas por WhatsApp, utilizando embeddings semánticos
y recuperación de contexto con ChromaDB y Cohere.

---

## Requisitos

- Python 3.10 o superior
- Cuenta y API Key de Cohere

---

## Instalación

```bash
git clone https://github.com/tuusuario/challenge-final-ecommerce-api.git
cd challenge-final-ecommerce-api
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
